# 🚀 YatraBook — Travel Booking Platform

A full-stack, production-level travel booking platform for India. Search and book **Trains**, **Flights**, and **Buses** with a premium, futuristic dark-themed UI.

> Built with React, Node.js, Express, MongoDB, Tailwind CSS, and Framer Motion.

## ✨ Features

- **Multi-modal Search** — Trains, Flights, Buses with real Indian routes
- **Smart Filters** — Price, duration, departure time, class
- **Booking Simulation** — Seat selection, payment flow, PNR generation
- **Waitlist System** — Automatic waitlist when seats are full
- **Recommendations** — Cheapest and fastest routes across all transport modes
- **Multi-city Planner** — Plan trips across multiple cities
- **User Dashboard** — Booking history, recent searches, profile
- **Premium UI** — Dark theme, glassmorphism, Framer Motion animations

## 🛠 Tech Stack

| Layer      | Technology                        |
|------------|-----------------------------------|
| Frontend   | React 18 + Vite + Tailwind CSS   |
| Animations | Framer Motion                     |
| Backend    | Node.js + Express                 |
| Database   | MongoDB Atlas (Mongoose)          |
| Auth       | JWT + bcrypt                      |
| Deployment | Vercel (frontend) + Render (backend) |
